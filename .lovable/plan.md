## Overview
A PWA (Progressive Web App) for basketball players in Indonesia, starting with the login/sign-up screen matching your uploaded design, and player profile & stats features ‚Äî UI only for now.

## Screens & Features

### 1. Login/Sign-Up Screen (Landing)
- Full-screen basketball court photo as background (using your uploaded image)
- Two rounded white buttons at the bottom: **Login** and **Sign Up**
- Dark overlay at the bottom for button readability
- Matches the exact layout from your screenshot

### 2. Login Page
- Email and password input fields
- "Forgot password?" link
- Login button
- Link back to Sign Up

### 3. Sign Up Page
- Name, email, password fields
- Sign up button
- Link back to Login

### 4. Player Profile Page
- Player avatar, name, and position
- Basic stats display: games played, points per game, assists, rebounds
- Edit profile capability

### 5. Stats Dashboard
- Visual stats cards showing key performance metrics
- Simple charts for tracking performance over time

## PWA Setup
- Installable from the browser to the home screen
- Proper manifest with basketball app icon and theme colors
- Mobile-optimized viewport and meta tags
- Offline-capable with service worker
